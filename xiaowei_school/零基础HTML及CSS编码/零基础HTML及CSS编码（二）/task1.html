《嫌疑人X的献身》的牛逼之处，在于它用的居然是一个古典得不能再古典的诡计，古典到四十年前的横沟正史就已经称之为“刚发生事件之时，读者就知道犯人是谁了”的地步（《黑猫酒店杀人事件》）。但东野利用书中的情节与人物刻画，很巧妙地包装了这个诡计，使得读者依然会被陷进自己的圈套中。换行了
全都事无巨细地抖露了出来，读者怎么想也想不出到底还有哪能安排一部“推理小说”该有的那些诡计什么的。然后，主角登场，查案进行，但读者的注意力全都被转移到中间的爱情故事与两位大神的学术切磋去了。文章链接：如何评价《嫌疑人X的献身》？然而，当读者翻到结尾，几乎已经忘记这是部推理小说的时候，作者突然笔锋一转，Yukawa老师告诉你，这个故事不仅有诡计，而且还是如此惊悚和可怕的一个诡计！而这个诡计设计得又的确是那么的巧妙，让读者感受到实实在在的震撼。

这本书的伟大还在于塑造了一个风头完全压过侦探的反派，无论在书里还是电影中，电影里绝对的演技派堤真一更是完胜偶像派福山雅治。推理小说中，凶手将侦探本人算入诡计中的写法，是极需要胆量的。迄今来读过的书中，只有艾勒里·奎因的《十日惊奇》来过这一道。新窗口打开：如何评价《嫌疑人X的献身》？而《嫌疑人X》里，反派石神的算计，恰恰就是把代表传统侦探高大形象的汤川学，放到了自认为最坚不可摧的一环：哪怕侦探知道了一切，他也必须做出抉择，不说出真相，眼前一切如故；但说出真相，他将亲手毁掉一切，甚至包括他自己，侦探本人将同样成为罪人。作者中间一面写女主角感情的顺利进展，一面写两位大师的学术讨论，都是上面这个算计的铺垫。读到Yukawa老师最后的那段推理时，作为读者真不得不为这个算计叫绝。
